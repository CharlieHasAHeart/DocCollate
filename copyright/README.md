# doccollate-copyright

A standalone CLI for generating software copyright application DOCX from JSON or specification files.

## Quick start

```bash
PYTHONPATH=src python3 -m doccollate_copyright run --input-json ./input_copyright.json --debug
```

Old style is still supported:

```bash
PYTHONPATH=src python3 -m doccollate_copyright --input-json ./input_copyright.json --debug
```
